# 水库水情分析规则

## 核心字段

| 字段 | 含义 | 单位 | 典型范围 |
|------|------|------|----------|
| rz | 库水位 | m | 视水库而定 |
| inq | 入库流量 | m³/s | 0-数千 |
| otq | 出库流量 | m³/s | 0-数千 |
| w | 蓄水量 | 万m³ | 视水库而定 |

## 分析规则

### 1. 水位变化率

```
最新两次采集: z1(新), z2(旧)
变化率 = (z1 - z2) / z2 * 100%

if |变化率| > 1%:
  → 关注: 水位变化较快
if |变化率| > 3%:
  → 预警: 水位变化异常
```

详见 `algorithms/water-level-change.md`

### 2. 库容平衡分析

```
if inq > otq:
  → 入库大于出库，水位应上升
  if rz 下降:
    → 异常: 可能存在蒸发/渗漏

if inq < otq:
  → 出库大于入库，水位应下降
  if rz 上升:
    → 异常: 数据可能有误

平衡偏差 = |实际水位变化 - 理论水位变化|
if 偏差 > 阈值:
  → 标记数据异常
```

### 3. 10分钟流量汇总

```
查询最近10分钟内所有设备数据
按设备ID分组取最新一条
分别对 otq(出库) 和 inq(入库) 求和
```

### 4. 异常值识别

```
if rz == null or rz == -1:
  → 数据缺失，显示 "-"
if inq < 0 or otq < 0:
  → 流量为负，数据异常
if rz 超出历史极值:
  → 超限预警
```
