# 水位变化率算法

## 计算公式

```
最新两次采集: z_new, z_old
变化量 = z_new - z_old
变化率 = 变化量 / z_old * 100%
```

## 水位涨幅预警

```
查询每个测站最新的两条记录
计算变化率，保留4位小数

if |变化率| > 1%:
  → 水位变化较快
if |变化率| > 3%:
  → 水位变化异常
if |变化率| > 5%:
  → 水位急剧变化，紧急关注
```

## 10分钟流量汇总

```
查询最近10分钟内所有设备数据
按设备ID分组，取每组最新一条
sum_inq = sum(各设备 inq)
sum_otq = sum(各设备 otq)
```

## 库容平衡校验

```
理论水位变化 ≈ (inq - otq) * 时间间隔 / 库容曲线
实际水位变化 = z_new - z_old
偏差 = |实际变化 - 理论变化|

if 偏差 > 阈值:
  → 可能存在蒸发、渗漏或数据误差
```
