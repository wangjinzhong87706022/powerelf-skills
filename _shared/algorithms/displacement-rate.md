# 位移速率算法

## 计算公式

```
查询近两年数据，按测点地址分组

速率 = (最大值 - 最小值) / 月份数 * 100 (单位: cm/月)

分别计算:
  speedX = X方向速率
  speedY = Y方向速率
  speedH = H方向速率

变化率差 = 当年速率 - 去年速率
```

## 速率等级

| 速率 | 等级 | 含义 |
|------|------|------|
| < 0.5 cm/月 | 稳定 | 正常范围 |
| 0.5-2 cm/月 | 缓慢 | 持续监测 |
| 2-5 cm/月 | 中速 | 需关注 |
| > 5 cm/月 | 快速 | 高风险 |

## 年度位移统计

```
Redis缓存: key = "getDeltaListStats{year}", TTL = 30分钟

按测站分组:
  H/X/Y 值取绝对值，保留2位小数
  最大值 = max(测点值)
  最小值 = min(测点值)
  平均值 = mean(测点值)
  年变幅 = 最大值 - 最小值
```
