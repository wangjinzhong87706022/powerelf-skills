# 智能插值规则

## 核心算法

自适应策略选择插值法。详见 `algorithms/interpolation-strategies.md`。

## 策略选择流程

```
输入: 含 null 的时间序列 data[]，有效数据点 validPoints[]

1. 统计 nullCount 和有效数据
2. if 无有效数据 → 返回原数据（无法插值）

3. 计算指标:
   - R² = 线性回归决定系数（越接近1线性越好）
   - a = 二次拟合 y=ax²+bx+c 的二次项系数
   - 周期性 = 自相关函数峰值检测

4. 策略选择:
   ┌─────────────────────────────────────────────────────┐
   │ nullCount < 3           → 线性插值（短缺失优先简单）│
   │ R² > 0.9 且 |a| < 0.05 → 线性插值（线性特征明显）  │
   │ R² > 0.7 且 |a| < 0.2  → 二次插值（轻度非线性）    │
   │ 周期性明显              → 样条插值（保留周期特征）  │
   │ 其他                    → 滑动平均（保守策略）      │
   └─────────────────────────────────────────────────────┘
```

摒弃原始实现中只有线性/二次两种策略的选择逻辑。

## 四种插值策略

### 1. 线性插值 (LinearFiller)

```
首尾缺失: 用最近有效值填充（前向/后向填充）
中间缺失: y = y0 + (y1 - y0) * (x - x0) / (x1 - x0)
精度: 3位小数，HALF_UP 舍入
```

### 2. 二次插值 (QuadraticFiller)

```
中间缺失: 找最近3个有效点，解方程组 y = ax² + bx + c
求解: LU 分解
降级: 有效点不足3个或求解失败 → 降级为线性
精度: 3位小数
```

### 3. 样条插值 (SplineFiller) — 新增

```
适用: 周期性数据（如水位日变化、温度日变化）
方法: 三次自然样条（Cubic Natural Spline）
优势: 保证一阶、二阶导数连续，曲线平滑
实现: 构建三对角矩阵，追赶法求解
```

### 4. 滑动平均 (MovingAverageFiller) — 新增

```
适用: 波动大、无明显趋势的数据
窗口: 5个有效点（前后各取最近的有效值）
公式: y = mean(窗口内有效值)
优势: 稳健，不受极端值影响
```

## 置信度评估 — 新增

```
confidence = R² * (1 - nullCount / totalCount) * (validCount / totalCount)

if confidence >= 0.8 → 高置信度，直接使用
if 0.6 <= confidence < 0.8 → 中置信度，标记为"建议复核"
if confidence < 0.6 → 低置信度，标记为"需人工复核"，不自动写入
```

摒弃原始实现中无条件写入数据库的做法。

## 数据增补流程

```
1. 查询昨日未增补的缺失记录
2. 按设备+业务表分组（避免重复）
3. 获取采集频率 freq
4. 构建逻辑位置列表: position = (hour * 60 + minute) / freq
5. 构建含 null 的输入列表（大小 = 1440 / freq）
6. 执行策略选择 + 插值
7. if 置信度 >= 0.6:
     写回业务表 + 更新缺失记录的 filledDataContent
   else:
     标记 needsReview = true，不写入
```
