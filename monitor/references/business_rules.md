# 业务规则参考

本文档列出监控分析引擎的所有阈值、等级划分和判定规则。Agent 执行分析时按此文档判定。

---

## 水库水情规则

### 水位变化率阈值

| |变化率| | 级别 | 颜色 | 动作 |
|---------|------|------|------|
| <= 1% | 正常 | 绿色 | 常规记录 |
| > 1% | 关注 | 蓝色 | 持续监测，记录变化 |
| > 3% | 预警 | 橙色 | 人工确认，加密监测 |
| > 5% | 紧急 | 红色 | 立即上报，启动响应 |

### 库容平衡判定

| 条件 | 结论 | 说明 |
|------|------|------|
| inq > otq 且 rz 上升 | 正常 | 入库大于出库，水位应上升 |
| inq > otq 且 rz 下降 | 异常 | 可能存在蒸发/渗漏，需核查 |
| inq < otq 且 rz 下降 | 正常 | 出库大于入库，水位应下降 |
| inq < otq 且 rz 上升 | 异常 | 数据可能有误，需核查 |
| inq = otq 且 rz 不变 | 正常 | 平衡状态 |

### 水位异常值判定

| 条件 | 结论 |
|------|------|
| rz = NULL | 数据缺失，显示 "-" |
| rz = -1 | 数据缺失/传感器故障 |
| inq < 0 或 otq < 0 | 流量为负，数据异常 |
| rz 超出历史极值 | 超限预警 |

## GNSS 变形规则

### 位移速率等级

| 速率 (cm/月) | 等级 | 颜色 | 建议动作 |
|--------------|------|------|----------|
| < 0.5 | 稳定 | 绿色 | 常规监测，每季度分析 |
| 0.5 - 2.0 | 缓慢 | 蓝色 | 持续监测，每月分析 |
| 2.0 - 5.0 | 中速 | 橙色 | 加密监测频率，现场核查，提交报告 |
| > 5.0 | 快速 | 红色 | 立即上报，启动应急预案，人员撤离评估 |

### 位移方向判定

| 条件 | 方向 | 含义 |
|------|------|------|
| wgs84_total_x > 0 | 向下游偏移 | 坝体向下游方向位移 |
| wgs84_total_x < 0 | 向上游偏移 | 坝体向上游方向位移 |
| wgs84_total_h > 0 | 下沉 | 坝体沉降 |
| wgs84_total_h < 0 | 上升(隆起) | 坝体隆起 |
| wgs84_total_y > 0 | 向右岸偏移 | — |
| wgs84_total_y < 0 | 向左岸偏移 | — |

### 一致性检查规则

| 观测现象 | 结论 | 建议动作 |
|----------|------|----------|
| 同断面多测点偏移方向一致 | 整体滑动风险 | 升级告警等级，启动专项监测 |
| 相邻测点偏移方向相反 | 局部变形 | 关注局部区域，现场核查 |
| 单测点偏移，周围稳定 | 个体异常 | 检查测点设备状态 |

### 年度位移统计

- 统计表: `srm_gnss_stat_day`
- 年变幅 = 年度最大值 - 年度最小值
- 分 H(垂直)、X(水平纵向)、Y(水平横向) 三个方向

## 闸泵工况规则

### 闸门流量合理性

| 条件 | 结论 | 可能原因 |
|------|------|----------|
| 开启高度=0 且 流量>0 | 异常 | 闸门漏水，密封失效 |
| 开启高度>0 且 流量=0 | 异常 | 闸前堵塞，传感器故障 |
| 开启孔数 > 总孔数 | 异常 | 数据录入错误 |
| 状态=关 且 开启高度>0 | 数据不一致 | 状态与开度矛盾 |
| 状态=开 且 开启高度=0 | 数据不一致 | 状态与开度矛盾 |

### 泵站电气参数范围

| 参数 | 正常范围 | 放宽范围 | 单位 | 说明 |
|------|----------|----------|------|------|
| 电压(U) | 361-399 | 340-420 | V | 标准380V±5%/±10% |
| 频率(f) | 49-51 | 48-52 | Hz | 标准50Hz±1%/±2% |
| 三相不平衡度 | < 5% | < 10% | % | max\|差值\|/均值 |

### 三相不平衡度计算

```
mean_current = (ia + ib + ic) / 3
不平衡度 = max(|ia-ib|, |ib-ic|, |ia-ic|) / mean_current * 100%
```

| 不平衡度 | 级别 | 动作 |
|----------|------|------|
| < 5% | 正常 | — |
| 5% - 10% | 注意 | 记录，关注趋势 |
| > 10% | 预警 | 检查线路，排查单相负载 |

### 负载率分析

| 负载率 | 级别 | 动作 |
|--------|------|------|
| < 10% (运行中) | 空载 | 检查是否需要停机 |
| 10% - 95% | 正常 | — |
| > 95% | 过载 | 降载或停机检查 |

### 冷却系统规则

| 条件 | 结论 | 动作 |
|------|------|------|

| 温升(出水-进水) > 10度 | 冷却不足 | 检查冷却水流量 |

### 励磁系统规则

| 条件 | 结论 |
|------|------|
| 励磁电压=0 且 设备运行 | 励磁系统异常 |

## 趋势检测规则

### Mann-Kendall 趋势判定

| S统计量 | 趋势 | 显著性判定 |
|---------|------|-----------|
| S > 0 | 上升趋势 | \|S\| > 临界值 → 显著 |
| S < 0 | 下降趋势 | \|S\| > 临界值 → 显著 |
| S ≈ 0 | 无明显趋势 | — |

临界值 (95%置信度): `1.96 * sqrt(n*(n-1)*(2*n+5)/18)`

### 变化点检测阈值

- 阈值 = 序列标准差 * 0.3 (默认比例)
- 左右均值差 > 阈值 → 标记为变化点
- 相邻变化点间距 < 5 个采样点 → 合并

### 周期性检测

- ACF > 0.3 → 该滞后处存在显著自相关
- 取 ACF 峰值对应的 lag 作为主周期

## 预测参数建议

### 指数平滑 (Holt-Winters)

| 指标 | alpha | beta | gamma | 季节周期m | 预测步长 |
|------|-------|------|-------|----------|---------|
| 水位 | 0.2 | 0.05 | 0.2 | 24(小时) | 1-6小时 |
| 渗压 | 0.15 | 0.03 | 0.15 | 24(小时) | 1-24小时 |
| GNSS位移 | 0.1 | 0.02 | 0.1 | 168(周) | 1-7天 |
| 流量 | 0.3 | 0.1 | 0.2 | 24(小时) | 1-6小时 |

### ARIMA 参数建议

| 指标 | p | d | q | 说明 |
|------|---|---|---|------|
| 水位 | 2-4 | 1 | 1-2 | 日变化有周期性 |
| 渗压 | 3-5 | 1 | 1-2 | 缓慢变化 |
| GNSS | 2-3 | 1-2 | 1 | 需要差分消除趋势 |
| 流量 | 3-6 | 1 | 2-3 | 波动较大 |

### LSTM 参数建议

| 参数 | 值 | 说明 |
|------|-----|------|
| seq_length | 96 | 输入序列长度 (24h * 4次/h) |
| pred_length | 24 | 预测长度 (6h * 4次/h) |
| hidden_size | 64 | 隐藏层大小 |
| num_layers | 2 | LSTM层数 |
| dropout | 0.2 | Dropout率 |
| learning_rate | 0.001 | 学习率 |
| batch_size | 32 | 批大小 |
| epochs | 100 | 训练轮数 |

## 算法选择指南

| 场景 | 推荐算法 | 理由 |
|------|----------|------|
| 实时预测(秒级) | 指数平滑 | 计算快，无需训练 |
| 短期预测(小时级) | ARIMA | 捕捉趋势和周期 |
| 中期预测(天级) | LSTM | 长期依赖 |
| 缺乏历史数据 | 指数平滑 | 不需要训练数据 |
| 多变量预测 | LSTM | 可处理多输入特征 |
| 需要置信区间 | ARIMA | 自带置信区间 |
