# 缺陷预测规则

## 预测方法

### 1. 线性趋势预测

适用于：短期（1-3个月）缺陷数量预测

```
模型：y = a * x + b
  x = 月份序号（0, 1, 2, ...）
  y = 当月缺陷数量

预测：y_future = a * x_future + b

解读：
  a > 0.5  → 缺陷上升趋势，需关注
  a < -0.5 → 缺陷下降趋势，治理有效
  |a| <= 0.5 → 基本平稳
```

### 2. 季节性分析

适用于：识别缺陷的周期性规律

```
水利场景常见周期：
- 汛期（6-9月）：水位变化大，闸门/泵站缺陷增多
- 冬季（12-2月）：低温导致设备故障率上升
- 定期检查后：缺陷集中发现期

分析方法：
  1. 按月聚合缺陷数量
  2. 计算每月平均值
  3. 识别峰值月份
  4. 提前部署检修
```

### 3. 设备缺陷热点预测

```
基于历史频率的贝叶斯预测：

P(设备i下月有缺陷) = (设备i历史缺陷次数 + 1) / (总月数 + 设备总数)

高风险设备判定：
  P > 0.3  → 高风险，建议预防性维护
  P 0.1-0.3 → 中风险，加强巡检
  P < 0.1  → 低风险，常规巡检
```

## 预警规则

### 缺陷数量异常增长

```
条件：本月缺陷数 > 上月缺陷数 × 1.5
级别：WARNING
动作：分析原因，可能是设备老化或巡检标准变化
```

### 新增高频缺陷对象

```
条件：某设备本月缺陷次数 >= 3
级别：WARNING
动作：建议专项检查或更换
```

### 缺陷处理积压

```
条件：未处理缺陷数 > 20 且 持续增长
级别：CRITICAL
动作：组织专项整改
```
